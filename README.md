#Moive App

##Technologies
Project is created with:

- React
- React-router-dom v6
- Axios
- Redux
- Redux-Thun
- Swiper
- Scss

##Setup
To run this project, install it locally using npm:

```
$ git clone https://github.com/DenisBukva/movies-shows-app.git
$ cd ../movies-shows-app
$ npm install
$ npm start

```
